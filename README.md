# シェア旅

## サイト概要
### サイトテーマ
実際に現地に足を運んだ旅人が、人におすすめしたいと思ったグルメや宿、観光スポットをシェアするコミュニティサイト

### テーマを選んだ理由
ガイドブックに載っているメジャーなスポットはもちろんのこと、それ以外の場所や情報も得らるようなサイトがあれば、より旅行が楽しくなるのではないかと考え、このテーマにしました。

### ターゲットユーザ
旅行に行く人、旅行好きの人

### 主な利用シーン
・旅行のプランを立てる時、旅先でのお店選びに困った時に検索して調べることができる。  
・自分の旅の記録としても活用できる。
